# node_plex_faketuner

Minimal HDHomeRun-compatible fake tuner for Plex Live TV.

This project allows Plex to treat any IPTV or FFmpeg-compatible stream as a
real Live TV tuner. Plex handles buffering, transcoding, and client delivery.

No Plex plugins required.

---

## Requirements

- Node.js 18+
- FFmpeg installed and available in PATH
- Plex Media Server
